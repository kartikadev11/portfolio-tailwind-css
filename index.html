Ecogreen

The Interactive Education App Recycling Program Based on Artificial Intelligence (AI) as a Learning and Business Opportunity. This is my project as a team. My task in this project are develop and deploy an image classification model to classify the type of waste using MobileNetV2, a sentiment analysis model using SVM, create a website dashboard consisting of UI, CRUD, and endpoint API using Bootstrap and Flask

IKost Website: Decision Support System for Selecting Boarding Houses Using the SAW Method

IKost is a website that provides information on boarding houses and places to help recommend boarding houses based on the criteria of price, facilities, location, room size, and security provided. This website was created using the Laravel framework, bootstrap, adminLTE. The IKost website provides Login and CRUD features, where admins can add, edit, and delete boarding house data, criteria, weights, and types of criteria. Users can view boarding house information and decision support system features to choose boarding houses by entering weights according to their needs.

ReBin: Recycle Your Bin

A Recycling Recommendation Application. This is my project as a team. My task in this project are build an image classification model to classify the type of waste using DenseNet121, a waste recycling recommendations model using a content-based filtering approach, utilizing the TF-IDF algorithm and cosine similarity, and create an endpoint API to deploy the model.
